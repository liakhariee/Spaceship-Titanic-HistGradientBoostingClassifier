# Spaceship-Titanic-HistGradientBoostingClassifier

## О проекте
Классификация. Соревнование Spaceship Titanic. Бинарная классификация (Transported or not).

## Структура проекта
- spaceship-titanic-histgradientboostingclassifier.ipynb - ноутбук с обучением и настройкой модели

Произведено сравнение алгоритмов классификации машинного обучения. Финальным выбран HistGradientBoostingClassifier.

Оптимизирована метрика Accuracy c помощью библиотеки Optuna.

## Метрики качества
- Accuracy: 0.78933

Dataset - https://www.kaggle.com/competitions/spaceship-titanic/data
